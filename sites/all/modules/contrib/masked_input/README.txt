Download the 3rd-party dependency library from
http://cloud.github.com/downloads/digitalBush/jquery.maskedinput/jquery.maskedinput-1.3.js
to the libraries directory of your site ("sites/all/libraries" or equivalent).

More information *might* be found in the Advanced Help topics. If it's not, I
encourage you to complain about it here: http://drupal.org/node/1365028. You
may also check out this blog (http://digitalbush.com/projects/masked-input-plugin/)
for an explanation of how the Masked Input jQuery plugin is used.
